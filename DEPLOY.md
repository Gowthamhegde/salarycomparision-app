# Railway Deployment Guide

## Quick Deploy Steps

1. **Push to GitHub**:
   ```bash
   git add .
   git commit -m "Clean deployment setup"
   git push origin main
   ```

2. **Deploy on Railway**:
   - Go to https://railway.app
   - Click "New Project"
   - Select "Deploy from GitHub repo"
   - Choose your repository
   - Add PostgreSQL database service
   - Set environment variables:
     - `DATABASE_URL` (auto-generated by Railway PostgreSQL)
     - `JWT_SECRET` (generate a secure random string)
     - `NODE_ENV=production`

3. **Database Setup**:
   - Railway will automatically run `prisma generate`
   - Run `prisma db push` to create tables
   - Import data using the admin route: `/api/admin/import-csv`

## Environment Variables Needed:
- `DATABASE_URL` - PostgreSQL connection string (auto-generated)
- `JWT_SECRET` - Random secure string for JWT tokens
- `NODE_ENV=production`
- `PORT` - Auto-set by Railway

## Project Structure:
- Server-only deployment (client removed for simplicity)
- Static files served from `server/public/`
- API routes under `/api/`
- Database: PostgreSQL with Prisma ORM

The app will be available at your Railway-generated URL.
